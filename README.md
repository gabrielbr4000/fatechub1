# Fatec Hub
A Flutter project about a communication app foccused on solving a Fatec misscommunication problem.

# How it works
It is an open-source mobile application built with the Flutter Framework, developed by two students for study purposes.

# Features
The app will have several features, including: 
- Activities and Internship Board;
For students to receive notifications about events at the university and notifications about internship opportunities.

- Messenger; 
To improve academic communication between students and teachers.

- Academic Calendar;
To help students better plan for exam and event dates.

- College Map;
To help students know where the classrooms for their enrolled subjects are located.

- Class Schedule;
To give students quick and easy access to their enrolled class schedule.

- User Settings;
For users to set up their account (changing their image or other information).

- Document Request.
To facilitate and expedite the process of students requesting documents from the academic secretariat.
